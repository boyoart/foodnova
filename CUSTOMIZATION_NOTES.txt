# FoodNova Customization Guide

## 1. LOGO CUSTOMIZATION

### Navbar Logo (Main Header)
File: `/frontend/src/components/Navbar.jsx` (Lines 24-31)

Current code uses a styled div with letter "F":
```jsx
<div className="w-10 h-10 bg-primary rounded-lg flex items-center justify-center">
  <span className="text-white font-bold text-xl">F</span>
</div>
```

To use your own logo image:
1. Place your logo file in `/frontend/public/logo.png`
2. Replace the code with:
```jsx
<img 
  src="/logo.png"
  alt="FoodNova" 
  className="h-10 w-auto"
/>
```

### Footer Logo
File: `/frontend/src/pages/Home.jsx` (Footer section around line 217)
The footer has text-only branding "FoodNova". Update the text or add an image similarly.

## 3. DELIVERY OPTIONS

The app supports two delivery types:

### Pickup (Free)
- Customer selects "Pickup" at checkout
- No delivery fee
- Shows store pickup location
- Order address marked as "PICKUP - ..."

### Delivery (Fee paid on delivery)
- Customer selects "Delivery" at checkout
- Enters delivery address
- Delivery fee is calculated based on location
- Customer pays delivery fee upon receiving order
- Orange notice displayed: "Delivery fee will be calculated based on your location and paid upon delivery"

### To Update Pickup Location:
File: `/frontend/src/pages/Checkout.jsx` (around line 200)

```jsx
<p className="text-sm text-muted-foreground mt-1">
  FoodNova Store<br />
  123 Main Street, Lagos, Nigeria<br />
  Open: Mon-Sat 8AM - 8PM
</p>
```

## 5. CURRENCY

All prices display in Nigerian Naira (₦)
Format function in: `/frontend/src/api/store.js`

```js
export const formatPrice = (amount) => {
  return `₦${amount.toLocaleString()}`;
};
```

To change currency symbol, update this function.
